FreeCodeCamp API Basejump: Image Search Abstraction Layer

User stories:

  • I can pass a search string to the url and set the number of skipped results by adding ?offset=10.
  • I can see the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can get a list of the most recently submitted search strings by passing /imagesearch to the url.

Example usage:

https://img-search-abstraction-roky.herokuapp.com/lol cats?=10
https://img-search-abstraction-roky.herokuapp.com/imagesearch
